## Bike Rental Dashboard 
```
Dashboard ini digunakan untuk menganalisis data penyewaan sepeda menggunakan Streamlit.
```
## Deskripsi
```
Project ini memiliki tujuan sebagai berikut:
- Menganalisis tren penyewaan sepeda berdasarkan hari, jam, musim, dan cuaca
- Mengidentifikasi pengaruh windspeed terhadap penyewaan sepeda
- Menyediakan visualisasi yang menarik
```
## Setup Environment
### 1. Clone Repository 
git clone https://github.com/sabibu12/Submission-Bike-Sabia-Putri-Aulia
### 2. Anaconda 
```
conda create -n main-ds python=3.13.12
conda active main-ds
pip instal -r requirements.txt
```
### 3. Run Streamlit App
```
dashboard
streamlit run dashboard.py
```
